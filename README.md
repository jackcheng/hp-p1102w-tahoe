# HP LaserJet Professional P1102w Driver for macOS 26.x (Tahoe)

## Overview

This repository provides a community-modified driver package for the **HP LaserJet Professional P1102w** printer, enabling compatibility with **macOS 26.x (Tahoe)**.

The official HP driver for this model does not natively support macOS Tahoe. This repackage addresses that by updating the `Distribution` file within the latest official HP driver, allowing for successful installation and operation on newer macOS versions.

## Features

*   **macOS 26.x (Tahoe) Compatibility:** Print seamlessly from your macOS Tahoe machine with your HP LaserJet Professional P1102w.
*   **Based on Latest Official HP Driver:** Ensures you're getting the most up-to-date core driver components from HP, with only the necessary modifications for OS support.

## Installation

1.  **Download:** Download the latest `HP_P1102w_macOS_Tahoe_Driver.pkg` (or similar name) from the [Releases section](https://github.com/jackcheng/hp-p1102w/releases) of this repository.
2.  **Run Installer:** Double-click the downloaded `.pkg` file to launch the installer and follow the on-screen prompts.
3.  **Add Printer:** After installation, go to `System Settings` > `Printers & Scanners` and add your HP LaserJet Professional P1102w printer.

**Important Note:** As this is a modified driver, you may need to approve the installation from an "unidentified developer" in your `System Settings` > `Privacy & Security` panel if prompted.
